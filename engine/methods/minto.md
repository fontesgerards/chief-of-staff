# minto — pyramid principle for written output

> Use for any written recommendation, memo, or update the principal will send or read.

- **Lead with the answer.** The governing thought / recommendation goes first, not last.
- **Group supporting arguments** under it — MECE, 3±1 groups.
- **Each level answers the reader's next question** ("why?" / "how?").
- **SCQA framing** when setting up: Situation → Complication → Question → Answer.
- **Bottom line up front (BLUF).** A busy reader who stops after the first paragraph still gets the decision.

Pair with `core/voice.md` for tone/length and `procedural/drafting.md` for learned habits.
